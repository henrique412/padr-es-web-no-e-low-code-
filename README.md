# 🐾 Neighborhood Pet Grooming - Presença Digital para Pequenos Negócios

Este projeto foi desenvolvido como parte de um desafio técnico voltado à criação de soluções digitais para pequenos negócios locais. O objetivo foi construir uma aplicação web funcional, utilizando ferramentas **no-code/low-code**, mas mantendo o rigor técnico dos padrões fundamentais da web (HTML, CSS e JS).

## 🚀 O Projeto
O **Neighborhood Pet Grooming** é uma Landing Page otimizada para um pet shop de bairro. A aplicação permite que o estabelecimento exiba seus serviços de forma dinâmica e capture agendamentos através de um formulário estruturado.

### 🛠️ Tecnologias e Ferramentas
*   **Webflow:** Plataforma principal para construção da interface e lógica de design.
*   **Webflow CMS:** Utilizado como banco de dados visual para gerenciamento de serviços e preços.
*   **CSS Avançado:** Utilizado para posicionamento de elementos e estilização personalizada.
*   **Make.com (Opcional/Sugestão):** Estrutura preparada para automação de notificações via e-mail ou WhatsApp.

## 🧠 Desafios Técnicos e Soluções

### Customização com Código (CSS/HTML)
Devido às limitações de planos gratuitos em ferramentas no-code, apliquei conhecimentos de **CSS nativo** para criar componentes que não estavam disponíveis na biblioteca padrão:
*   **Botão de WhatsApp Flutuante:** Criado manualmente utilizando propriedades de `Position: Fixed` e `Z-Index`, garantindo que o canal de conversão esteja sempre visível ao usuário, independentemente da rolagem.
*   **Box Model:** Todo o layout foi estruturado respeitando as propriedades de *margin*, *padding* e *border*, garantindo um design limpo e profissional.

### Padrões Web e Boas Práticas
*   **Responsividade:** O site foi otimizado para Desktop, Tablet e Mobile através de breakpoints customizados.
*   **Acessibilidade:** Aplicação de tags semânticas (H1, H2, H3) e preenchimento de *Alt Texts* em todas as imagens para suporte a leitores de tela.
*   **Ética Digital:** Inclusão de termos de privacidade no formulário de contato, garantindo transparência no tratamento dos dados dos clientes.

## 📂 Estrutura do Site
1.  **Hero Section:** Apresentação da marca com Call to Action (CTA).
2.  **Catálogo de Serviços:** Seção dinâmica alimentada pelo CMS.
3.  **Sobre Nós:** Breve histórico e valores do negócio.
4.  **Agendamento:** Formulário integrado para captação de leads.

## 📈 Aprendizados
Este projeto reforçou a ideia de que o uso de ferramentas no-code exige um entendimento profundo da arquitetura da web. Contornar limitações técnicas da ferramenta escolhida exigiu um pensamento analítico focado em como o navegador interpreta o CSS e o HTML, resultando em uma aplicação robusta e escalável.
